# Chickpea Gene Expression Atlas v2.0

## Overview

The Chickpea Gene Expression Atlas v2.0 is an interactive web-based resource developed for visualization and exploration of gene expression patterns across diverse chickpea tissues and developmental stages. The atlas integrates transcriptomic datasets with interactive visualization tools to support functional genomics, candidate gene validation, molecular breeding, and crop improvement research.

The platform provides:

- Interactive gene expression atlas (eFP-style visualization)
- Tissue-specific expression profiling
- Expression heatmaps
- Expression bar plots
- GWAS candidate gene search
- Editable chromosome mapper
- Downloadable publication-quality figures

---

## Features

### Gene Expression Atlas

Visualize tissue-specific expression patterns using a custom SVG-based chickpea tissue diagram.

- Gene search by gene ID
- Interactive tissue coloring
- Adjustable expression scale
- Custom color palettes
- Downloadable figures

### Expression Heatmaps

Generate expression heatmaps for single or multiple genes.

- TPM-based expression values
- log2(TPM + 1) transformation
- Tissue-specific subsets
- Publication-ready visualization

### Expression Profiles

Explore expression patterns using interactive bar plots.

- Individual gene visualization
- Tissue group filtering
- Raw TPM and log-transformed values

### GWAS Candidate Gene Explorer

Search genes located within genomic intervals surrounding significant SNPs.

- Chromosome-based search
- Adjustable upstream/downstream windows
- Candidate gene export

### Chickpea Chromosome Mapper

Interactive chromosome visualization module.

- Multiple chickpea reference versions
- Chromosome-specific visualization
- Custom marker placement
- Marker start and end coordinates
- Editable marker colors
- Downloadable chromosome maps

## Transcriptomic Analysis Workflow

An independent transcriptomic analysis pipeline was performed for this resource.

### Data Processing

1. RNA-seq reads retrieved from NCBI SRA
2. Quality assessment using FastQC
3. Adapter trimming and filtering using Trimmomatic
4. Transcriptome indexing using Salmon
5. Transcript quantification using Salmon quasi-mapping
6. Gene-level TPM matrix generation
7. Averaging of biological replicates
8. Visualization and atlas construction

### Reference Genome

CDC Frontier Chickpea Reference

Source:

https://data.legumeinfo.org/Cicer/arietinum/
