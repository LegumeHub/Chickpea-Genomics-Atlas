About the Chickpea Gene Expression Atlas

The Chickpea Gene Expression Atlas is an interactive web resource developed to visualize gene expression patterns across 32 chickpea tissues and developmental stages. The resource allows users to search individual genes, view tissue-specific expression using an eFP-style atlas, generate expression bar plots, create heatmaps for selected genes, summarize mean and median expression, identify candidate genes near GWAS-associated SNPs, and create editable chickpea chromosome maps.

This resource is designed to support candidate gene prioritization, transcriptomic interpretation, seed trait-related functional genomics, GWAS visualization, and molecular breeding research in chickpea.

Credit and Data Source

The RNA-seq data used in this resource were obtained from the publicly available chickpea transcriptome dataset deposited under NCBI BioProject PRJNA622231. This dataset includes transcriptomic profiles from 32 tissues and developmental stages of chickpea cultivar ICC4958 and served as the source of the raw sequencing data used for the analyses presented in this atlas.

Reference:
Jain M, Bansal J, Rajkumar MS, Garg R. An integrated transcriptome mapping the regulatory network of coding and long non-coding RNAs provides a genomics resource in chickpea. Communications Biology. 2022;5:1106. PMID: 36261617.

Transcriptomic Analysis

Publicly available RNA-seq data from BioProject PRJNA622231 were used to examine expression profiles of candidate genes associated with chickpea seed traits. While the original study generated and released the RNA-seq datasets, all downstream transcriptomic processing, expression quantification, data integration, visualization development, and web-resource implementation were performed independently for this project.

Raw sequencing reads were downloaded from the NCBI Sequence Read Archive using the SRA Toolkit and converted into paired-end FASTQ files. Transcript-level abundance was estimated using Salmon v1.10.2. A Salmon index was generated from the CDC Frontier reference transcriptome file. Salmon quantification files were imported into R using tximport, transcript-level estimates were summarized to gene-level TPM, and biological replicates were averaged within each tissue to generate tissue-level mean TPM expression matrices.

Website Development

The atlas website was developed using a custom SVG-based chickpea tissue diagram and gene-level expression matrix. Tissue-specific SVG elements were linked to expression values, allowing dynamic coloring of tissues based on selected gene expression.

The website also includes modules for bar plots, heatmaps, mean and median expression summaries, downloadable figures, GWAS candidate gene searches, and an editable chickpea chromosome mapper based on reference chromosome lengths.
