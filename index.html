About the Chickpea Gene Expression Atlas

The Chickpea Gene Expression Atlas is an interactive web resource developed to visualize gene expression patterns across 32 chickpea tissues and developmental stages. The resource supports gene search, eFP-style visualization, expression plots, heatmaps, GWAS candidate gene searches, and editable chromosome maps.

Credit and Data Source

The RNA-seq data used in this resource were obtained from the publicly available chickpea transcriptome dataset deposited under NCBI BioProject PRJNA622231.

Transcriptomic Analysis

Raw RNA-seq reads were downloaded from the NCBI Sequence Read Archive, quality-checked, trimmed, and quantified using Salmon against the CDC Frontier reference transcriptome. Transcript-level estimates were summarized to gene-level TPM, and biological replicates were averaged within each tissue to generate tissue-level mean TPM expression matrices.
